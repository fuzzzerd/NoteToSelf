*Someone [asked on Game Development](https://gamedev.stackexchange.com/q/11159):*

> Does anyone have experience using SQLite with XNA?
> 
> Or using any other DB with XNA?

*I posted the following answer, which received 5 upvotes:*

You should have no issues Accessing SQLite from a Windows XNA Game. On Windows, XNA Project is just like any other project and you can do anything that C# can do. On Xbox or WP7 this isn't completely the case; however, there is also a C# version of SQLite written to run on fully managed environment (Xbox, WP7): see the ZMan's post here: [http://xboxforums.create.msdn.com/forums/p/47127/282261.aspx](http://xboxforums.create.msdn.com/forums/p/47127/282261.aspx)
